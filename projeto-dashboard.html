Dashboard de Suporte de TI

Status: Em desenvolvimento

Visão Geral

Este projeto nasceu da minha experiência direta com suporte técnico. É um sistema de tickets (CRUD - Create, Read, Update, Delete) projetado para ser simples e intuitivo, tanto para o usuário final que abre um chamado quanto para o técnico que o gerencia. A ideia é centralizar a comunicação, organizar as demandas e fornecer métricas básicas sobre os atendimentos.

Tecnologias Utilizadas

  • Front-End: HTML5, CSS3, JavaScript (com possível uso de um framework como React ou Vue no futuro)
  • Back-End: Node.js com Express.js (planejado)
  • Banco de Dados: SQL (PostgreSQL) ou NoSQL (MongoDB) (planejado)
  • Versionamento: Git e GitHub
